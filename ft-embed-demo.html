⚽ Embed Football Schedule on Your Site

Add a live football match schedule widget to your website for free. The widget updates automatically — no registration needed.

⚠️ Required attribution The embed code below includes a mandatory link to yosintv.net. Do not remove it — the widget will stop working if that link is missing from your page.

Copy & paste this code into your website

<!-- ⚽ Football Schedule Widget by YoSinTV — required attribution link below, do not remove -->
<iframe
  id="yosintv-ft"
  src="https://www.yosintv.net/ft-embed.html"
  width="100%"
  height="500"
  style="border:none;border-radius:10px;overflow:hidden;display:block;"
  title="Football Schedule – YoSinTV"
  loading="lazy"
></iframe>
<script>
(function(){
  var iframe = document.getElementById('yosintv-ft');
  var hasLink = !!document.querySelector('a[href*="yosintv.net"]');
  iframe.addEventListener('load', function(){
    iframe.contentWindow.postMessage(
      { yosintvAuth: hasLink },
      'https://www.yosintv.net'
    );
  });
})();
</script>

Widget data is sourced from YoSinTV Football Schedule. Match times are shown in your visitor's local timezone automatically.
